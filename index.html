Featured Projects

  • Pokedex

    • Link: https://github.com/J9NIC3/pokedex.git
    • Pokedex is a project-based learning experience where we are tasked with creating a Pokémon search engine using Python. This project involves taking user input and implementing corresponding display functions. As an additional feature, we included a surprise section that was not part of the initial assignment. This extension allowed us to deepen our understanding of Python, enhancing our proficiency in the language. Moreover, working on this project taught us valuable communication skills as we collaborated with our partners, exchanging different perspectives and assisting each other in identifying potential program errors."

  • My Emotional FlashCards

    • Link: https://github.com/J9NIC3/emotions-flashcards.git
    • My Emotional Flashcards is a project developed by my friends and me during the Youth Digital Currency Programme (YDCP), using basic HTML,CSS and JavaScript. The goal of this project is to digitize learning materials for young children. These flashcards are designed to teach children about various emotions, enhancing their emotional intelligence. We integrated sound effects corresponding to different expressions to create a more engaging and interactive learning experience.

  • Pizza Hut Learning Website

    • Link: https://github.com/J9NIC3/pizza.git
    • The Pizza Hut Learning Website was a project I worked on during my participation in the Youth Digital Currency Programme (YDCP). This website was created using HTML and CSS to design the interface of a Pizza Hut delivery website. During the development process, I applied the knowledge I had acquired previously, utilizing various HTML tags and CSS styles to replicate the actual interface of the delivery website. This experience provided me with a comprehensive understanding of the core concepts underlying webpage development.
